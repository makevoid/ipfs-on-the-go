# ipfs-on-the go

---


this app provides you IPFS when you need it

- upload a file
- select time range (now to 1 month, now to 1 year, now to 2 years)
- pay 0.1 1 or 2 mbtc



### to run this app:


### prerequisite: `ruby`

### setup: `gem i bundler && bundle`

### run: `bundle exec rackup -p 3002 -o 0.0.0.0`
